# Icon files (Lucide SVGs)

Place here one SVG file per icon name. The app loads them from `/media/icons/{name}.svg`.

**Expected filenames (must match exactly):**

- `accessibility.svg` – Accesible filter
- `non-binary.svg` – Neutral filter
- `baby.svg` – Cambiador filter
- `droplets.svg` – Higiene íntima filter
- `star.svg` – Favoritos
- `lock.svg` – Solo clientes filter

Download the SVG from [Lucide Icons](https://lucide.dev/icons) (e.g. "Copy SVG") and save with the name above.
